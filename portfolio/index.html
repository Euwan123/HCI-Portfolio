Final Project

Campus Navigator Mobile Application

HCI Final Project

An interactive mobile prototype designed to help students locate campus facilities, view real-time availability of study spaces, and access wayfinding assistance with minimal cognitive load.

Project Objectives

  • Reduce navigation time for new and returning students
  • Apply usability heuristics to improve interface clarity
  • Validate design decisions through user testing sessions
  • Deliver an accessible prototype aligned with WCAG guidelines

Role in the Project

Lead UI/UX Designer and Frontend Prototyper — responsible for user research, wireframing, high-fidelity mockups, and usability evaluation.

Technologies Used

Figma HTML5 CSS3 JavaScript Miro
